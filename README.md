# Privacy Policy — Amplemarket Toolkit

**Last updated: May 22, 2026**

## Overview

Amplemarket Toolkit is an internal Chrome extension built exclusively for use by the Amplemarket team. This policy explains how the extension handles data.

## Data Collection

Amplemarket Toolkit **does not collect, transmit, or share any personal data** with third parties. No analytics, tracking, or telemetry of any kind is performed.

## Data Stored Locally

The extension stores the following data **locally on your device only**, using the browser's built-in chrome.storage API:

- Your API key for internal incident reporting
- Your display name and user ID for reporter attribution
- Escalation history (ticket references you created)
- Extension preferences and settings

This data never leaves your device and is never sent to any server other than the internal services the extension is explicitly designed to interact with.

## Third-Party Services

The extension communicates only with internal company services as part of its core functionality, initiated explicitly by the user. No data is shared with any advertising, analytics, or unrelated third-party service.

## Contact

For questions about this policy, contact your internal tools administrator.
